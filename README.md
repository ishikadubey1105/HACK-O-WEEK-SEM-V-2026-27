## Week 1 — Library Management System REST API

> A clean REST API for managing books, members, and borrowing operations, built with **Flask** and **SQLite**.

📁 Folder: [`Week-1/`](./Week-1/)

### Features
- Full CRUD for Books and Members
- Borrow & Return with 14-day loan tracking
- Overdue detection
- Beautiful pastel frontend (no framework — plain HTML/CSS/JS)
- Postman collection included

### Quick Start
```bash
cd Week-1
pip install -r requirements.txt
python run.py
# Visit http://127.0.0.1:5000/
```
